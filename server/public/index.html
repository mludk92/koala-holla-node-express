Quick Run through of funtionality

  • Input values into the 5 fields and press Add Koala button
    to update the db and DOM with new values
  • Values are Editable on the DOM with use of ContentEditable, cannot update db
  • If input ready for tranfer is true, creates button ready to transfer
    • on click of ready to tranfer, updates value to transfered, on DOM and db
    • prompted by sweetalert to confirm click
  • Delete Button to remove item from db and dom by db table index
    • promted by sweetalert to confirm choice
  • input field, and dom element which are drop downs hold valid inputs for the field
    • db requires boolean value so if drop down values are not used, the CRUD opertaions will not complete
    • click in this field will clear the value and allow for new input incase of typing mistake
  • update edits button is not functional at this point, req.body was {} and ran out time to research
    how to have these values sent in the response to CRUD operations
  • there are two search fields to narrow down results
    • search by name or gender, just start to type the value you are looking for
